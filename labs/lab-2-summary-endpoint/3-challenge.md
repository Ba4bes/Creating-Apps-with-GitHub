# Lab 2 — Challenge Self-Directed: Validation, Tests, and Agent Comparison

**Goal:** Harden your API with validation, write a test, and compare your work with automated output.

**Time:** 25 minutes

**You will need:** Lab 2 Core completed (summary endpoint working).

---

## Your Task

Use Copilot Chat to review your `POST /entries` endpoint for validation gaps and fix what it finds. Write at least one test for the summary endpoint using your language's test framework. If a coding agent PR is available, compare its approach with yours.

## Success Criteria

- POST endpoint has improved input validation
- At least one test file exists and passes (`npm test`, `pytest`, or `dotnet test`)
- You can describe one difference between your implementation and an automated one (if a coding agent PR is available)
