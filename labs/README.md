# Lab Instructions

Every lab has two tiers and two modes:

| | Core | Challenge |
|---|---|---|
| **Guided** | Step-by-step instructions with expected output | Step-by-step stretch goal |
| **Self-directed** | Goal and constraints only | Goal and constraints only |

Pick your mode at the start of each lab. You can switch at any time. Both modes produce the same outcome.

## Format

Each file opens with:
- **Goal** — one sentence describing what you will accomplish
- **Time** — suggested duration
- **You will need** — prerequisites for this lab
