# Lab 1 — Core Self-Directed: Create Copilot Instructions & Prompt Files

**Goal:** Create Copilot configuration files that accurately describe your working style and this repo's conventions, then verify they shape output.

**Time:** 35 minutes

**You will need:** Lab 0 completed, Codespace open, your language folder chosen.

---

## Your Task

Create a `copilot-instructions.md` that tells Copilot about your language choice, code style preferences, and testing approach for this repo. Create at least one prompt file for a task you will do in Lab 2 (implementing an API endpoint).

Then use inline completion (not chat) to write something small and compare the output quality to what you would get without the instructions file.

## Success Criteria

- `.github/copilot-instructions.md` exists with Language, Code style, and Testing sections
- `.github/prompts/feature.prompt.md` exists with useful implementation guidance
- You have used inline completion and observed Copilot following your instructions
- Zero premium requests consumed

> **Important:** What you write now is what Copilot uses in Lab 2.
