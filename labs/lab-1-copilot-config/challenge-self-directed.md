# Lab 1 — Challenge Self-Directed: Extend Your Copilot Configuration

**Goal:** Build out a complete Copilot configuration for this repo and measure the difference it makes.

**Time:** 25 minutes

**You will need:** Lab 1 Core completed (instructions file and feature prompt file exist).

---

## Your Task

Add a `test.prompt.md` that describes how tests should be structured for your chosen language. Add a review checklist to your `copilot-instructions.md`. Optionally use Copilot Chat to compare output quality with and without your prompt files attached.

## Success Criteria

- `.github/prompts/test.prompt.md` exists with framework-specific testing guidance
- `copilot-instructions.md` includes a Review checklist section
- You can articulate what difference the configuration files make to Copilot's output
